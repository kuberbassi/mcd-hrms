# MCD HRMS - Project Documentation

## What is this?
This is the HRMS website for MCD. It helps the office manage staff, salary, and attendance without paper files.

> **Note**: For login passwords, check the **[README.md](./README.md)** file.

---

## Problems We Fixed

| Problem | Our Solution |
| :--- | :--- |
| **Old Registers** | We made online attendance. |
| **Salary Calculation** | The system does the math for us. |
| **Phone Use** | The site works well on mobile. |
| **Can't See Data** | Staff can login to check their own records. |
| **Permissions** | We fixed the access errors. |
| **Hiring** | HR can now post jobs easily. |


---

## Main Features

### For Admin & HR
*   **Dashboard**: See total staff count.
*   **Hiring**: Post jobs and check applications.
*   **Staff**: Add or remove employees.
*   **Attendance**: Mark Present/Absent.
*   **Workforce**: Managing Payroll and Ratings.
*   **Requests**: Approve transfers and check complaints.
*   **Tasks**: Assign work.

### For Employees
*   **Profile**: Check details.
*   **Tasks**: Finish assigned work.
*   **Attendance**: Check history.
*   **Salary**: Check monthly slip.
*   **Requests**: Ask for transfer or complain.

### For Everyone
*   **Careers**: Apply for jobs without login.
